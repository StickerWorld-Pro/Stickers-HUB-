
# 🔥 Sticker Hub PRO

Sticker Hub PRO is a modern, responsive sticker website where users can browse, search, preview, like, and download high-quality stickers easily.

## 🚀 Features

- 🔍 **Search Stickers** – Quickly find stickers using the search bar  
- 📂 **Categories** – Browse Anime, Meme, and Movie stickers  
- 👁 **Preview Mode** – View stickers in full-screen popup  
- ⬇ **Download Option** – Download stickers instantly  
- ❤️ **Like System** – Like stickers and track popularity  
- 🌙 **Dark Mode** – Switch between light and dark themes  
- 📱 **Responsive Design** – Works smoothly on mobile and desktop  

## 🛠 Built With

- HTML  
- CSS  
- JavaScript

## 📌 How to Use

1. Browse stickers or use the search bar  
2. Click **View** to preview  
3. Click **Download** to save stickers  
4. Like your favorite stickers
